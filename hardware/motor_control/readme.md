# motor_controller 模块说明

## 项目背景
骨科居家康复终端，ELF 2 开发板（RK3588），Ubuntu Desktop 系统。
我是同学C，负责硬件驱动模块。

## 我的任务（第二周）
用 periphery 库控制 GPIO → MOS驱动 → 震动马达，
实现三种震动模式：短震 / 长震 / 急促震。

## 硬件说明
- GPIO 电压：0~3.3V，严禁超压
- 引脚编号公式：n×32 + (字母序号-1)×8 + 数字，A=1 B=2 C=3 D=4
- 当前占位引脚：GPIO3_B3 = 107，硬件到货后修改 GPIO_PIN 常量
- 驱动链路：GPIO → MOS模块 → 5V电源 → 马达，必须共地
